## What
Every scene of *Akira 1988* as a gif. Browse and share, or just watch an [endless stream](https://gifakira.com/stream.html).

## Building
Scripts use webpack and TypeScript:

```bash
$ npm install
$ webpack 
```

Site uses [Jekyll](https://jekyllrb.com):

```bash
$ jekyll serve
```
